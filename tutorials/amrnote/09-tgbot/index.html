Telegram-бот предназначен для быстрого и удобного доступа к схемам терапии. С его помощью пользователи могут:

  • Просматривать актуальные схемы лечения прямо в чате, без необходимости переходить на сайт;
  • Скачивать схемы в формате PDF для удобного использования в офлайн-режиме или для печати;
  • Получать доступ к информации в любое время и из любого места через мобильное устройство или компьютер.

Выбрав команду "/help" - вы увидите списко доступных вам действий.

/teams - отображает список команд в которых вы состоите.

/public - отображает список публичных схем.

/about - отображает инцформацию о боте.

Структура Telegram бота организована следующим образом:

  • Команда - начальная точка, из которой пользователь выбирает нужную схему терапии.
  • Схема - каждая схема может содержать вложенные и связанные элементы (доступна для скачивания в PDF):
    • Дочерняя схема
    • Связанная схема (доступна для скачивания в PDF)
    • Терапия (доступна для скачивания в PDF)
    • Инфоблок (доступен для скачивания в PDF)
