Мои препараты

В разделе Мои препараты содержится информация о препаратах, которые используются в ваших схемах терапии.

Вы можете либо самостоятельно создать препарат, нажав на кнопку «+» и введя его название.

Также вы можете импортировать препарат с сайта AMRbook. Для этого нажмите на кнопку «+», затем «Импортировать из AMRbook», выберите из выпадающего списка необходимый препарат и нажмите кнопку «Сохранить». Лекарственное средство с описанием будет импортировано в список ваших препаратов Мои препараты.

Для объединения нескольких препаратов в один существует специальный инструмент «Объединение». Для объединения необходимо указать название нового препарата (которое будет отображаться после слияния) и выбрать те препараты, которые необходимо объединить. Текстовые описания всех объединяемых препаратов будут просто записаны последовательно друг за другом.

Теги

Для добавления тега к препарату необходимо нажать на значок «+» и либо выбрать существующие теги из списка, либо ввести название нового. Подробнее о тегах см. главу «Теги».

Элементы управления

Рассмотрим подробнее все элементы управления, представленные на панели описания препарата.

  • Заполнить из AMRbook — позволяет импортировать данные о препарате с сайта AMRbook в уже созданный препарат.
  • Печать — позволяет распечатать или сохранить в PDF-файл информацию о препарате. Распечатанный/сохранённый документ возможно будет удобно использовать врачам вашего отделения/клиники при отсутствии доступа к Интернету.
  • Изменить — позволяет изменить название препарата.
  • Редактировать описание — позволяет изменить текстовое поле «Описание» препарата.
  • Удалить — позволяет удалить выбранный препарат.
