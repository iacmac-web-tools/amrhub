Сборник — это тематическая подборка схем терапий, объединённых по какому-либо признаку: по болезни, по клинической ситуации, по целевой аудитории (например, взрослые, дети) или по назначению (экстренная помощь, амбулаторное лечение и т.д.).

Основные характеристики:

  • Название сборинка
  • Список схем
## Панель сборники

Панель сборников включает в себя следующие элементы:

  1. Выпадающий список, отоюбражающий названия сборников.
    2. При выборе сборника отображается иерархический список схем, которые он содержит. По умолчанию отображает все схемы из всех сборников. Если сборник публичный, то слева от названия отображается специальный значок.
  2. Кнопка с дополнительными действиями
    3. Меню этой кнопки зависит от текущего состояния выбора:

    Если выбран пункт "Все сборники":

    • Добавить сборник - создаёт новый пустой сборник с указанным названием.

    Если выбран конкретный сборник, то отображается:

    1. Добавить сборник
    2. Редактировать схемы в сборнике - открывает режим управления содержимым.
    3. Переименовать - позволяет изменить название сборника.
    4. Поделиться/сделать приватным - переключает режим доступа.
    5. Показать ссылку - доступно, если сборник публичный. Отображает URL для просмотра.
    6. Удалить - полностью удаляет сборник.
