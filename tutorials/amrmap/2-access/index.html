2. Доступ к системе

2.1 Общий доступ

AMRmap - это онлайн платформа анализа данных резистентности к антимикробным препаратам в России, которая содержит набор инструментов для визуализации данных о чувствительности микроорганизмов к антимикробным препаратам и распространенности основных генетических детерминант устойчивости к антибиотикам. Система представляет собой кроссплатформенное веб-приложение, расположенное по адресам: map.antibiotic.ru и AMRmap.ru. Перейдя по указанной ссылке пользователь попадает на главную страницу приложения:

В верхней части экрана расположены навигационные вкладки (ниже на рис. обведены красным), переключаясь между которыми, пользователь может выбрать интересующий раздел приложения:

На вкладке Главная пользователю доступна возможность выбора уровня доступа.

Приложение предоставляет пользователям два уровня доступа:

  • общий доступ;
  • экспертный доступ (данные для доступа рассылаются персонально участникам проекта).

Уровень доступа пользователя определяет его возможности при работе с приложением. Все пользователи по умолчанию имеют общий уровень доступа. Пользователи с экспертным уровнем доступа имеют возможность просматривать дополнительные показатели внутри города, результаты собственных исследований в рамках проектов, в которых принимали участие, в остальном возможности пользователей с экспертным и общим доступом совпадают.

2.2 Экспертный доступ

Для участников многоцентровых проектов НИИАХ и МАКМАХ онлайн-платформа AMRmap предоставляет возможность получения и анализа результатов исследования собственных изолятов в центральной лаборатории НИИАХ (г.Смоленск).

Для входа в приложение с экспертными правами доступа необходимо на вкладке Главная нажать кнопку Экспертам (ниже на рис. обведена красным цветом).

После этого действия появится всплывающее окно Экспертный доступ, где пользователю необходимо ввести Имя пользователя и Пароль, а затем нажать кнопку Принять (ниже на рис. обведена красным цветом).

В случае успешного входа и получения экспертного доступа приложение сообщит об этом пользователю. Чтобы выйти из открывшегося окна регистрации необходимо нажать Закрыть (ниже на рис. обведена красным цветом).

После закрытия всплывающего окна Экспертный доступ пользователь попадает на вкладку Главная, где, в случае успешного входа и получения экспертного доступа, отображается сообщение и появляется кнопка Выйти (см. рис. ниже). Чтобы пользователю с экспертным доступом выйти из приложения и получить права общего доступа необходимо на вкладке Главная нажать кнопку Выйти (на рис. ниже обведена красным).

После получения прав Экспертного доступа, можно переходить к интересующим вкладкам приложения и приступать к анализу данных.

С целью получения информации по результатам собственных исследований в рамках проведенных проектов пользователю с экспертным уровнем доступа, кроме пунктов рассматриваемых в главе руководства Анализ данных, необходимо выбрать параметр 1.1 Проект и 5. ЛПУ в поле Параметры основные.

Для того, чтобы увидеть данные по всем проектам, в которых было принято участие (например, Марафон-2013 – Марафон 2016) следует последовательно выбрать эти проекты из выпадающего списка 1.1 Проект

Для просмотра результатов по необходимому ЛПУ в выпадающем списке 5. ЛПУ следует выбрать подходящий вариант.

Подробная информация по другим подвкладкам и возможностям онлайн-платформы AMRmap представлена в остальных главах руководства.

Важно!!! При просмотре результатов проведенного анализа пользователю с экспертным уровнем доступа отображается информация по ID ЛПУ в разделах МПК тренд (подраздел Таблица).
